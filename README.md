# PiicoDev® Magnetometer MMC5603 MicroPython Module


This is the firmware repo for the [Core Electronics PiicoDev® Magnetometer MMC5603](https://core-electronics.com.au/catalog/product/view/sku/XXXXXX)

This module depends on the [PiicoDev Unified Library](https://github.com/CoreElectronics/CE-PiicoDev-Unified), include `PiicoDev_Unified.py` in the project directory on your MicroPython device.


See the [Quickstart Guide](https://piico.dev/pX)
